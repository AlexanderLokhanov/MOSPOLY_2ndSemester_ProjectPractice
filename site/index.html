Возможности

Всё, что нужно
для учёта посещаемости

Администратор запускает опрос одной командой — бот сделает остальное автоматически.

👑

Роль администратора

Пользователь с заданным Telegram ID имеет право запускать опросы. Остальные команды для него недоступны простым студентам.

📢

Массовая рассылка

По команде /poll бот моментально рассылает вопрос всем, кто хоть раз написал боту.

✅

Быстрая отметка

Одно нажатие — «Да» или «Нет». При отсутствии бот запрашивает причину и сохраняет её в базу данных.

📝

Фиксация причин

Студент вводит причину отсутствия текстом — бот ждёт ответа и записывает его к нужной отметке автоматически.

🗄️

База данных посещений

Все отметки хранятся в SQLite с привязкой к пользователю, времени опроса и причине отсутствия.

🔒

Приватность

Бот работает только с теми, кто сам написал первым. Данные хранятся локально, без сторонних серверов.

Как это работает

Три шага до результата

01

Напишите боту

Отправьте /start в Telegram — бот зарегистрирует вас и запомнит ваш chat_id для будущих рассылок.

02

Администратор запускает опрос

Администратор отправляет команду /poll — бот рассылает всем студентам вопрос: «Вы на паре?»

03

Нажмите кнопку

«Да» — присутствие отмечено. «Нет» — бот попросит указать причину. Всё займёт не больше 10 секунд.
